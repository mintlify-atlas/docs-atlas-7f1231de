## Overview

The Beaver Builder REST API plugin exposes Beaver Builder content through WordPress REST API endpoints, enabling seamless integration with remote applications, headless WordPress implementations, and third-party services.

This plugin provides programmatic access to retrieve rendered HTML, compiled CSS, and JavaScript for Beaver Builder layouts, making it easy to consume Beaver Builder content in any application.

## Use Cases

- **Headless WordPress**: Integrate Beaver Builder content into React, Vue, or Next.js applications
- **Mobile Apps**: Fetch page content and styling for native mobile applications
- **Remote Integration**: Access Beaver Builder layouts from external systems and services
- **Content Distribution**: Deliver Beaver Builder content across multiple platforms

## API Namespace

All endpoints are registered under the `fl-builder/v1` namespace:

```
https://your-site.com/wp-json/fl-builder/v1/
```

## Requirements

<Note>
  This plugin requires Beaver Builder to be installed and activated on your WordPress site.
</Note>

- WordPress 4.4 or higher (WordPress REST API support)
- Beaver Builder plugin (any version)
- PHP 5.6 or higher
